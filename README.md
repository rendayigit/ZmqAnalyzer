# ZmqAnalyzer

Postman for ZeroMQ.

This project has been migrated to Python for easier usage and portability.

## Requirements

- Python 3.x
- wxPython
- pyzmq

## Installation

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

Run the application:

```bash
python zmq_analyzer.py
```

Or use the provided script:

```bash
./scripts/run.sh
```
